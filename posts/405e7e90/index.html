安装Node.js

Node.js、npm、nvm

Node.js

Node.js发布于2009年5月，由Ryan Dahl开发，是一个基于Chrome V8引擎的JavaScript运行环境，使用了一个事件驱动、非阻塞式I/O模型，让JavaScript 运行在服务端的开发平台，它让JavaScript成为与PHP、Python、Perl、Ruby等服务端语言平起平坐的脚本语言。

Node.js对一些特殊用例进行优化，提供替代的API，使得V8在非浏览器环境下运行得更好，V8引擎执行Javascript的速度非常快，性能非常好，基于Chrome JavaScript运行时建立的平台， 用于方便地搭建响应速度快、易于扩展的网络应用。

npm

npm（node package manager），是JavaScript世界的包管理工具，并且是Node.js平台的默认包管理工具，通过npm可以安装、共享、分发代码，管理项目依赖关系。

nvm

nvm（Node.js Version Management），是一个Node.js的版本管理工具，通过安装它可以安装和切换不同版本的Node.js。

什么是Hexo

Hexo是使用Node.js开发的一个快速、简洁、高效、开源的静态博客生成器

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

Hexo有什么优点

  • Hexo非常小巧，使用nodejs开发，下载安装时是需要几个简单的npm命令即可
  • 纯静态博客，不需要什么额外的mysql数据库、php环境，只需要一个nodejs环境即可
  • 纯静态博客，部署方便
  • 支持markdown语法，兼容windows/linux/mac，高可扩展性和自定义，有非常多的主题
